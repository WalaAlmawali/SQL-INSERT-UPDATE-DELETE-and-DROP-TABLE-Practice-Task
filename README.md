# SQL-INSERT-UPDATE-DELETE-and-DROP-TABLE-Practice-Task

Task Description: SQL Data Management (INSERT, UPDATE, DELETE, DROP TABLE)
In this set of tasks, trainees will work with two real-world database systems: a Company Management System and a Hospital Management System. The purpose is to practice full database data lifecycle operations, starting from inserting data, modifying it, removing specific records, and finally deleting entire tables.
These tasks are designed to simulate how real databases are maintained and managed in professional environments.
Trainees will work with pre-defined tables and will apply SQL commands to perform different operations on existing data
